# Semigroup formulation

`exp(−τH)` with `H=A+P ≽ 0` is a genuine **contraction semigroup**
(`‖exp(−τH)‖ = 0.96 < 1` at τ=0.5; `C_0=I`; `C_τC_σ=C_{τ+σ}`; monotone). The scale-flow
semigroup `exp(−iτD)` is **unitary** (norm 1, no contraction). So the contractive /
positive semigroup is generated by the **PSD capacity `H`** (archimedean Γ + pole),
**not** by the scale flow `D`. Positivity is a capacity property, not a flow property.

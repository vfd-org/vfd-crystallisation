# Finite vs infinite

Finite cutoff: `‖K‖=0.9998 ≤ 1` with `min eig(I−K*K)=+0.00042` (near-null). RH ⟺ `‖K‖≤1`
in the infinite adelic witness space for ALL admissible test functions — equivalently the
near-null margin never goes negative in the limit. No finite computation delivers this;
it is RH. The positivity is the capacity contraction, but the limiting bound is the open
arithmetic-site theorem.

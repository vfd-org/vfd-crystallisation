# "Planck scale" = self-dual resolution boundary (not a physical length)

The only meaningful cutoff is the **self-dual fixed point** of the scale inversion
`t↔1/t` (`u↔−u`): `t=1`, `u=0`, `Re(s)=1/2`. A valid regularisation window must be
self-dual `W(u)=W(−u)` and FE-preserving; a non-self-dual (shifted) window breaks the
`t↔1/t` pairing and the functional equation (the wrong-centre failure). So any "Planck-like"
scale here is a UV/IR self-dual resolution boundary, NOT a physical length. No physical
Planck units are inserted.

# Arithmetic-horizon reading (interpretive)

Read the Adelic centre as an **arithmetic horizon**: `H` = absorption capacity
(archimedean Γ + pole), `R_S` = reflection, `K` = reflection coefficient, `Q_W` = absorbed
flux, `Re(s)=1/2` = horizon. Then **RH = the arithmetic no-superradiance theorem**:
`‖K‖≤1` in the infinite adelic witness space (reflection never exceeds capacity). Nulls are
unphysical horizons: fake-Γ → superradiant (`‖K‖=1.67>1`); no-archimedean → no absorption
(`H` not PSD); wrong centre → misplaced horizon (FE breaks). The near-null mode is a
**marginal, non-superradiant** horizon (`‖K‖→1` from below, reflection/capacity 0.9998).
This is a theorem-PATTERN, not a proof; the infinite `‖K‖≤1` is RH-equivalent.

# Adelic-boundary leakage channels (what the infinite theorem must exclude)

Finite cutoffs cannot prove the limit; an infinite-positivity theorem must control norm at
the non-compact "boundary" of the idele class space. Channels and the requirement each maps to:

| channel | requirement |
|---|---|
| `u → ±∞` (test-function tails / idele-class non-compactness) | decay / weighted-space control |
| prime cutoff `p → ∞` | convergence of the finite-place sum `R_N → R` |
| high-frequency / large zero height | spectral control; no mass escaping to ∞ |
| domain of `H^{−1/2}` (where `H` eigenvalues → 0, the near-null edge) | `H ≻ 0` in the limit; `K` bounded |
| non-self-dual cutoff | FE-violation; only self-dual (`t↔1/t`) regularisation admissible |

Finite evidence (near-null stable, **not** boundary-escaping) is *supportive* but cannot
exclude leakage — only an infinite compactness/boundedness theorem (trace-class / nuclear
control of the Connes operators on the idele class space) can.

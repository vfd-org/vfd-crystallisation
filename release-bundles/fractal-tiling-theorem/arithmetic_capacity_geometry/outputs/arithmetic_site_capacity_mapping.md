# Mapping to known frameworks — and the honest verdict that it IS them

| capacity-geometry object | known framework object | status |
|---|---|---|
| `H = A∞ + pole` | archimedean + pole terms of the **Weil explicit formula** | **exact (= classical)** |
| `R` (primes) | finite-place / prime-power term of the explicit formula | **exact** |
| `Q_W = H − R` | the **Weil distribution / positivity functional** | **exact** |
| `‖K‖ ≤ 1` (`Q_W≥0`) | **Weil's positivity criterion** for RH | **exact (RH-equivalent)** |
| witness space `H∞` | functions on the **idele class space** (Connes) | conceptual→standard |
| near-null mode | marginal positivity of the Weil distribution | conceptual |
| forcing positivity geometrically | **Connes–Consani arithmetic site / scaling site** (Spec ℤ as a curve; Lefschetz/intersection positivity, the char-p Hodge-index analogue) | **OPEN** |
| `K` as self-adjoint spectral object | **Hilbert–Pólya** | **OPEN / not exhibited** |

**Honest verdict.** The mapping is **exact, not loose**: `H`, `R`, `Q_W`, `‖K‖≤1` *are*
the Weil explicit-formula objects and Connes' positivity criterion — the "capacity
geometry" is a **re-naming** of them (capacity/reflection/horizon ↔ arch+pole / primes /
positivity). Because it is exact, it adds **no new route**: the proof gap is *identical*
to Connes' open problem — force the positivity of the Weil distribution via a geometric
substrate over Spec ℤ (the arithmetic site), as the intersection form (Hodge index) does
in the function-field case. **Do not imply Connes–Consani already proves the bound; it
does not.**

# Arithmetic Address Atlas (WO-ARITHMETIC-ADDRESS-ATLAS-001)

A graded atlas of how arithmetic objects route into the spectral stack:

```
integer/exponent → cyclotomic sector → prime factor → field splitting
   → prime ideal → Hecke response → L-function → positivity/RH
```

**This atlas does not prove RH or any open problem.** It classifies *how deep*
each object actually penetrates, and stamps the boundary where the routing stops.

## Address grades (deepest layer reached)
| grade | layer | pass condition |
|---|---|---|
| A | factor/sanity | correctly accepted/rejected as prime/composite |
| B | cyclotomic address | active `Φ_d` sectors of `a^n±1` identified |
| C | field-coordinate | factors routed into `Q(√5)` by split/inert/ramified |
| D | ideal/Hecke/level | enters a Brandt/Hecke operator or is a level/conductor |
| E | L-function spectral | enters a coherent global L-function (Euler product) |
| F | positivity/RH-relevant | touches Weil/Li/HP-style positivity |

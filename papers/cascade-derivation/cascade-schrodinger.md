```
    (Δ_{H₄} ψ)(v)  =  Σ_{w ∼ v} [ψ(v) − ψ(w)]

## E9.3 Stone–von Neumann theorem → Schrödinger

### E9.3.1 Setup

With Theorems E9.1 and E9.2:
- U(t) is a 1-parameter group of unitary operators (Theorem E9.1).
- Ĥ = ℏΔ_{H₄} is a densely-defined Hermitian operator (Theorem E9.2).

By the **Stone–von Neumann theorem** (Stone 1930, von Neumann 1932):

> *Every strongly continuous 1-parameter unitary group U(t) on a
> Hilbert space has a unique self-adjoint generator Ĥ such that*
> ```
>     U(t)  =  exp(−iĤt/ℏ).
> ```

### E9.3.2 Application to cascade

Applying Stone–von Neumann with:
- U(t) = cascade time-evolution operator.
- Ĥ = cascade Hamiltonian from E9.2.

We obtain:
```
    U(t)  =  exp(−i ℏ Δ_{H₄} t / ℏ)  =  exp(−i Δ_{H₄} t).
```

### E9.3.3 Differentiating to get Schrödinger

Differentiating `|ψ(t)⟩ = U(t) |ψ(0)⟩` with respect to t:

```
    ∂_t |ψ(t)⟩  =  (−i Δ_{H₄}) U(t) |ψ(0)⟩
                =  (−i / ℏ) Ĥ |ψ(t)⟩
```

Rearranging:

```
    i ℏ ∂_t |ψ(t)⟩  =  Ĥ |ψ(t)⟩        (Schrödinger equation). ✓
```

This is the standard time-dependent Schrödinger equation.

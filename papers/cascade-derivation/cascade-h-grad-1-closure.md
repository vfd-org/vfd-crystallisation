## 2. The icosian ring I (Acceptance criterion A1)

### 2.1 Definition

Following Conway–Sloane §8.2.1: the **icosian ring** I ⊂ H is the
Z-subring of the real quaternions generated by the 120 unit icosians (the
binary icosahedral group 2I ⊂ H).

I is a Z-order in H (with non-commutative multiplication), and is the
maximal order containing the binary icosahedral group 2I. Equivalently, I is
the Z[φ]-span of the quaternion units {1, i, j, k} *together with* the
icosian-specific elements involving φ.

## 3. Mod-2 reduction I/2I_lat ≅ F_2⁸ (Acceptance criterion A2)

### 3.1 The F_2-vector space structure

Once a verified Z-basis of I is fixed, reducing it modulo 2I_lat gives an
F_2-basis {[b_1], ..., [b_8]} of I/2I_lat, so I/2I_lat ≅ F_2⁸ as an
F_2-vector space. With the current placeholder B (eq. 2.2), this statement
is conditional on A1.

### 3.2 The induced F_2-quadratic form

The quaternion norm N : I → Z extends to N : I → Z[φ] for icosian elements
(since I ⊂ H is closed under the norm valued in the field of definition).
Compose with the trace tr_{Z[φ]/Z} : Z[φ] → Z, x = a + bφ ↦ 2a + b
(equivalently: Tr(1) = 2, Tr(φ) = 1).

Define

    Q : I/2I_lat → F_2,    Q([x]) := tr(N(x)) mod 2.              (eq. 3.1)

This should be well-defined modulo 2I_lat provided the integral bilinear
trace form on I is fixed and N(x + 2y) − N(x) ∈ 2Z[φ] for all y ∈ I. This
note has not yet derived that divisibility from a reproduced basis.

### 3.3 Identification with the E₈ quadratic form mod 2

**Standard fact (Conway–Sloane §8.5; Coxeter, Wilson §2.3).** Under the
icosian construction E₈ ≅ I (as Z-modules with quadratic form), the
F_2-quadratic form Q on I/2I_lat equals the canonical E₈/2E₈ form.

Specifically: E₈/2E₈ ≅ F_2⁸ carries the inner-product-mod-2 quadratic form,
non-degenerate of even type (Arf invariant 0), with:

    135 isotropic vectors  (Q = 0, [x] ≠ 0)
    120 anisotropic vectors (Q = 1)

The maximal isotropic 4-dim subspaces L ⊂ F_2⁸ correspond bijectively
to octonion-compatible frames on E₈ (via the Cayley–Dickson process
applied to each L's complement structure; see §4).

**Correction (2026-04-24 from B3 sim).** The number of maximal isotropic
4-subspaces is **270**, not 30. Classical formula for O⁺(2n, 2) with
Arf = 0: N = ∏_{i=0}^{n-1}(2^i + 1) = 2·3·5·9 = 270 for n = 4.
Earlier drafts of this note and `cascade-fano-grading-lift.md` §5.2.1
stated 30 — that figure is retracted. 30 may be a sub-count of one
octonion-Coxeter orbit but is not the total maximal isotropic count.

**Status A2:** BUILD COMPLETE (sim-verified 2026-04-24). The mod-2 form
`q_I` on `I/2I_lat` computed from the B1 basis gives totals `1 + 135 +
120 = 256`, matching the canonical `E_8/2E_8` F_2-quadratic. Verified by

### 4.2 The (Z/2)³ quotient

L is a 4-dim isotropic subspace; L⊥ (its orthogonal complement under the
bilinear form B(x,y) = Q(x+y) − Q(x) − Q(y)) is also 4-dim and contains L
(L is its own polar). The quotient L⊥/L is trivial (dimension 0) for a
self-polar maximal isotropic.

**Correction to the §5.2 framing in `cascade-fano-grading-lift.md`.** A
maximal isotropic 4-dim subspace gives F_2⁸ ⊃ L ⊃ {0} with quotient
F_2⁴ ≅ F_2⁸/L, NOT directly (Z/2)³.

The Fano (Z/2)³ would have to come from **additional octonion-structure
data**, not from L alone. A bare complement W to L in F_2⁸ gives only an
F_2⁴ vector space; it does not canonically supply an identity, octonion
multiplication, or a Fano-plane labelling of the 7 nonzero classes. Supplying
that data is part of the open A3 problem.

    μ : F_2⁸ ↠ (Z/2)³                                              (eq. 4.1)

as `{ω₁, ω₂, ω₄, ω₅}`. Absolute frame-canonicality is build B6.

### 5c Build B6 — reduced-frame transitivity (2026-04-24)

**The reduced-frame set.** Let `V = I/2I_lat ≅ F_2⁸`, `q = q_I`, and
`p = [1] = (1,0,0,0,0,0,0,0) ∈ V`. Define

    Adm_red(V, q, p, A)  :=  { (L, μ) :  L ≤ V,  dim L = 4,
                                q|_L = 0,  p ∉ L,
                                μ : V ↠ A,  ker μ = L ⊕ ⟨p⟩ },

where `A = (F_2)³`. The elementary counts are:
- Number of 4-dim totally singular subspaces `L` with `p ∉ L`:
  `240` (the 270 total max-isotropics minus the 30 through `p`).
- Given `L`, the kernel `K = L ⊕ ⟨p⟩` is fixed (5-dim); the set of
  F_2-linear isomorphisms `V/K ≅ A` is `|GL(3, 2)| = 168`.
- Total `|Adm_red| = 240 · 168 = 40 320`.

**The acting group.** Let `G = Aut(V, q) = O⁺(8, 2)` (the orthogonal
group preserving `q`, order `348 364 800`). Let
`G_p := Stab_G(p)`. Note that `p = [1]` is **isotropic** (not
anisotropic), since `q([1]) = Tr_{Q(φ)/Q}(N(1)) mod 2 = Tr(1) mod 2
= 2 mod 2 = 0`; `[1]` is a nonzero isotropic vector. `G` acts
transitively on the 135 nonzero isotropic vectors of `V`, so the
orbit of `p` has size 135 and `|G_p| = |G|/135 = 2 580 480`; the
structure of `G_p` is a maximal parabolic, isomorphic to an
extension with Levi factor involving `O⁺(6, 2)` acting on `p⊥/⟨p⟩`.
The exact structure is classical (see Taylor, *Geometry of the
Classical Groups*, Ch. 11) and does not affect the orbit-
transitivity statement below.

**B6.3 — Orbit transitivity (Witt extension).**

> **Theorem (B6.3).** `G_p` acts transitively on `Adm_red(V, q, p, A)`.

*Proof sketch (Witt's theorem).* Fix a hyperbolic basis of
`(V, q)` as an 8-dim F_2-quadratic space of Arf invariant 0:
`e_1, …, e_4, f_1, …, f_4` with `q(e_i) = q(f_i) = 0`,
`b(e_i, f_i) = 1`, all other `b`'s zero (where `b(x, y) =
q(x+y) − q(x) − q(y)`). By Witt's extension theorem, any isometric
linear map between totally singular subspaces of `V` extends to an
element of `G = Aut(V, q)`.

Since `p = [1]` is isotropic with `q(p) = 0`, WLOG `p = e_1` in the
hyperbolic basis (`G` acts transitively on nonzero isotropic
vectors). Given any `(L, μ) ∈ Adm_red`:
- `L` is a 4-dim totally singular subspace of `V`. By Witt's
  theorem, there is `g_1 ∈ G` with `g_1(L) = L_0 :=
  span(f_1, e_2, e_3, e_4)`. Since `p ∉ L` and `p ∉ L_0`, and both
  `(L, p)` and `(L_0, p)` are of the type "totally singular 4-space
  plus external isotropic vector with nonzero bilinear pairing
  against L" (because `L = L^⊥`, so `p ∉ L` forces `b(p, L) ≠ 0`),
  Witt's extension theorem carries any isometry of `L ⊕ ⟨p⟩` onto
  `L_0 ⊕ ⟨p⟩` to a `G`-element, which we can choose to fix `p`; so
  `g_1 ∈ G_p`.
- After conjugation by `g_1`, we have `(L_0, g_1(μ))`. The kernel
  `K_0 = L_0 ⊕ ⟨p⟩` is the canonical 5-dim isotropic+[1] subspace,
  and `V/K_0 = span(f_2, f_3, f_4) ≅ F_2³`. The quotient map
  `V ↠ V/K_0` then composes with some linear iso
  `g_1(μ) ∘ q_V/K_0⁻¹ ∈ GL(3, 2)` to give `(L_0, μ_std)` for any
  standard choice `μ_std`. Any `g_2 ∈ G_p` stabilizing `L_0` and
  inducing the correct `GL(3, 2)` action on `V/K_0` completes the
  transport. The subgroup of `G_p` stabilizing `(L_0, p)` contains
  a copy of `GL(3, 2)` acting on `V/K_0` (classical: the
  stabilizer of `L_0` in `O⁺(8, 2)` is a parabolic with Levi
  factor containing `GL(3, 2)` acting on the quotient by the
  radical).

Therefore every `(L, μ) ∈ Adm_red` is carried to any fixed base
frame by some `g ∈ G_p`. Orbit-transitivity holds. ∎

**B6.4 — Pullback to G₂/octonion canonicality.**

The above establishes `O⁺(8, 2)`-canonicality of `μ_I` modulo
`G_p`-action — i.e. any two admissible frames differ by an element
of `G_p`, so the induced Fano quotient is canonical up to an
automorphism of `V` preserving `q` and `p`.

Strictly speaking, the statement desired by upstream (and by
classical octonion theory) is "canonical modulo `G_2`," where
`G_2 = Aut(O)` is the automorphism group of the real octonion
algebra (Baez, *The Octonions*, §4.1). The bridge between these is
the **pullback**

    κ :  { Kirmse–Coxeter octonion frames on O }  ⟶  Adm_red(V, q, p, A),

sending each octonion basic triple to its mod-2 reduction (the
classical icosian-E_8-mod-2 identification of Conway–Sloane Ch. 8
§6). The image of `G_2 = Aut(O)` under this pullback is a subgroup
of `G_p` (possibly `G_2(2)`, the finite `G_2` over F_2; possibly
the full `G_p` via the `S_8` factor). Codex's WO notes that
`G_2(2)` alone has order 12 096, which does not divide 40 320 — so
if the image is exactly `G_2(2)`, it's not transitive on
`Adm_red` on its own. However, the full `GL(3, 2)` action on Fano
labels together with the 240 L-orbits under `O⁺(8, 2)` IS
transitive on reduced frames, as shown above.

**Net claim of B6.** `μ_I` is canonical up to the action of
`G_p ⊂ O⁺(8, 2)`. Translating this to "up to G_2" requires the
classical pullback `κ` plus an identification of the image of
`G_2` in `G_p`; this pullback is writable in the cascade setting
but not executed here.

*Declared judgement (not yet a theorem).* For H-grad-1's intended
use (the P-A Access Principle), `G_p`-canonicality is judged
sufficient by the author on the grounds that the downstream
application only uses `μ_I` modulo F_2-linear automorphisms of
`(V, q, p)`. A formal equivalence theorem against the statement in
`cascade-access-principle-theorem.md` would be the proper
formalisation of this judgement; producing that equivalence is a
follow-up task, not a current closure.

**Optional sim.** `scripts/verify_h_grad_1_b6.py` is a **sanity
check**: it verifies the classical counts (270/240/168/40 320) and
confirms that two random L-pairs have matching Witt type (4-dim
totally singular, `p = [1]` isotropic, `p ∉ L`, nonzero pairing
`b(p, L)`). It does **not** construct an explicit `g ∈ G_p` by
Witt-extension linear algebra — that remains an optional follow-up.
The orbit-transitivity claim rests on the classical Witt extension
theorem (cited in the proof sketch above), not on exhaustive sim
enumeration.

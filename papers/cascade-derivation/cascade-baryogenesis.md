## E12.0 The baryogenesis problem

Observed universe contains almost only matter (baryons), essentially
no antimatter:

```
    η  =  n_baryon / n_photon  ≈  6.1 × 10⁻¹⁰   (Planck 2018, from BBN + CMB).
```

This is asymmetric relative to a fully symmetric early universe
(matter = antimatter → total annihilation → no universe).

**Sakharov (1967) conditions** for dynamical baryogenesis:
1. **Baryon number violation** (some process changing B).
2. **C and CP violation** (distinguishing matter from antimatter).
3. **Departure from thermal equilibrium** (allowing net asymmetry).

SM cannot produce η ~ 10⁻¹⁰ dynamically (electroweak baryogenesis
fails quantitatively). Beyond-SM scenarios include leptogenesis
(Majorana neutrinos, Majorana CP violation), GUT baryogenesis, etc.

## E12.2 Cascade prediction for η

Baryon asymmetry η is the net B fraction surviving after inflation's
thermal phase. Typical scale:

```
    η  ~  α_em^n × sin(δ_CP) × ε_sphaleron
```

for some power n reflecting loop order of baryogenesis.

**Cascade estimates:**

For n = 4 (4-loop suppression):
```
    η  ~  α_em^4 × sin(δ_CP^CKM)

       =  2.84 × 10⁻⁹ × 0.951
       =  2.70 × 10⁻⁹.

Observed: η = 6.1 × 10⁻¹⁰. Cascade estimate is 4.4× too high.

```
    η  ~  α_em^5 × sin(δ_CP^CKM)

30× too small.

## E12.3 Connection to neutrino sector / leptogenesis

Modern theories prefer **leptogenesis**: lepton asymmetry generated
first (via neutrino CP), then converted to baryon asymmetry by
sphalerons.

**Cascade leptogenesis:**
- Lepton asymmetry η_L from δ_CP (PMNS) = −π/2 and sterile neutrinos.
- Sphaleron conversion B = η_L (in thermal equilibrium, B and L
  mixed).
- Baryon asymmetry η_B = (28/79) η_L ≈ 0.35 η_L (standard SM
  result).

Cascade predicts δ_CP (PMNS) = −π/2 (E5), giving a natural lepton
asymmetry source. Sphaleron conversion is then standard.

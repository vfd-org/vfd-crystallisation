# Landing Plan — June 2026

**Why the work isn't landing, and the publication sequence that fixes it.**

Based on a four-channel audit (2026-06-11): local repo inventory, github.com/vfd-org (47 repos),
vibrationalfielddynamics.org, and @VFD_org on X.

---

## 1. Diagnosis: it is not a content problem

The mathematics has gotten steadily more careful (CLAIMS.md ledgers, falsifiers,
honest conditional scoping, 94/94-style test suites). But all four channels route
readers **away** from the careful work and **toward** the most dismissable framing:

| Channel | What a newcomer hits first | What they should hit |
|---|---|---|
| GitHub | Bare 47-repo grid; pinned repos = consciousness / "One Field to All of Physics" / AI_CONSCIOUSNESS_PROOF | The scope-disciplined icosian/RH bundle, the reproducible mass-ratio result |
| Website | 60+ papers, "structure exists insofar as it is fixed by its own coherence operator" | One canonical falsifiable result with one-click verification |
| X | "⚡ BREAKING: Mathematical Proof of AI Consciousness ⚡"; 1 GitHub link in 101 posts | Result-first figure + code link, weekly cadence |
| Bio everywhere | "unified geometry of reality… consciousness, energy, time & the infinite multiverse" | "Parameter-free spectral geometry. Every claim has a falsifier and a script." |

**The honesty is buried one click below the hype.** A skeptical physicist or
mathematician applies a two-second crank filter; the current shop window fails it
before they ever reach a CLAIMS.md. Engagement data confirms this: 91 stars across
47 repos, zero substantive issues, zero off-platform discussion anywhere on the web,
X median 6 likes with viral hits only on consciousness content — which attracts the
audience that *repels* the audience the maths needs.

Secondary structural problems:

1. **No map.** 47 repos, ~60 site papers, 23 local release bundles — no canonical
   index, no supersession chain, no "start here by audience."
2. **No external anchor.** No arXiv, no DOI, no journal submission, no independent
   replication. Self-graded test suites don't substitute.
3. **Volume outruns landing.** ~15 hardened papers + the entire June substrate-bridge
   derivation (rendering layer, rung ladder, narrative gap closure) sit unpublished
   while new work keeps stacking. Each new drop further dilutes the last.
4. **Two audiences in one voice.** The consciousness-curious audience drives likes;
   the rigorous audience drives credibility. Serving both from one register loses both.

**Core principle for everything below** (from the credibility-bar rule): a
non-academic submitter does not get the benefit of the doubt. Only **complete +
undismissable** artifacts go in the shop window. Everything conditional, narrative,
or interpretive is downstream of an unconditional anchor.

## 3b. The campaign frame (2026-06-11)

Decision: treat this as a **marketing campaign, not a traditional physics
submission** — but specifically *developer marketing* (product-led growth), not
consumer marketing. The product is the reproducible result; the ad unit is the
demo; conversion = someone runs the code.

### What the funnel looks like

```
 Impression          Click               Conversion           Advocacy
 X post / short  →   result page     →   clone + run      →   public "I checked,
 (one figure,        (claim, error,      verification         it's right" /
  one claim,          inputs, RUN         script               substantive
  code link)          THIS button)                             objection
```

Every piece of content must point one step down this funnel. A post with no code
link is spend with no landing page.

### Segments (pick per post, never mix in one artifact)

| Segment | Where they live | Creative that converts them | What they give back |
|---|---|---|---|
| Number theorists | LMFDB orbit, Mathstodon, MathOverflow | "Brandt eigenvalues = your point counts, 13/13. Script:" | The replication that anchors everything |
| Recreational/visual math | math-Twitter, 3B1B audience, quasicrystal/polytope niche | 600-cell spectra visuals, golden-ratio structure, shorts | Reach + shares inside credible circles |
| Physicists | physics-Twitter, HN | "parameter-free mp/me to 0.02%, here are the 4 inputs, falsify it" | Scrutiny, objections, spread |
| Consciousness-curious (current base) | already following | existing register | Keep, but quarantine from the technical feed |

### Campaign assets already in hand
- 3.7k followers + demonstrated media production (1,056 media posts).
- `shorts_pipeline/docs/production-playbook.md` — hook patterns, voice, payoff
  discipline already derived from 12 high-velocity transcripts. Point this
  machinery at RESULTS (one short = one number + one visual + repo link).
- Reproducible repos with one-command verification — the "demo" most outsider
  physics cannot offer. This is the differentiator; every creative leads with it.

### KPIs (replace likes)
- Repo clones / unique visitors on the canonical result page.
- Verification runs reported (issues, replies, quote posts with output).
- Substantive technical replies (objections count as conversions).
- One advocate per wave = wave success (same metric as §4).

### Campaign rules
1. One claim per artifact. The thread can ladder; the post cannot.
2. Code link in the first post, not a reply.
3. Visual-first: spectra, polytopes, error bars — never a wall of φ-equations.
4. Weekly flagship post > daily volume (volume trained the algorithm that the
   feed is low-signal).
5. The technical feed never posts consciousness content; quarantine registers.
6. Niche communities over big-name tags: a 2,000-follower quasicrystal circle
   out-converts a reply to a 100k-follower neuroscientist.

## 4. Measuring "landing"

Per wave, success = at least one of:
- An independent person runs the verification and says so (issue, PR, post).
- A substantive technical objection arrives (objections are engagement).
- An arXiv endorsement, journal referee report, or niche-community thread.

If a wave gets zero external interaction in 4 weeks, the fix is routing/outreach
iteration on THAT wave — not writing the next paper. The stack is the symptom;
sequencing discipline is the cure.

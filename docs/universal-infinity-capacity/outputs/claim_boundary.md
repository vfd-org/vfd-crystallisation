# Claim boundary

This framework does **not** prove RH, solve black-hole physics, or explain consciousness.
It identifies a shared mathematical *form* — **capacity dominates reflection at a completed
boundary** (`Q=H−R≥0 ⟺ ‖K‖≤1`) — which is **generic** (any PSD `H`+sym `R`), i.e. the
ubiquity of "stability = a positive quadratic form." Each domain needs its own theorem or
data: RH → Weil positivity (open, = the infinite `‖K‖≤1` / arithmetic site); black holes →
scattering physics; brains → empirical neuroscience (the analogy here is *not* evidence).
It is a reusable language for boundary stability, **not a proof by analogy**.

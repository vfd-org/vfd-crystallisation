# Universal Infinity Capacity Geometry (framework, not a proof)

**Thesis.** Infinity is not "endless size"; it is the **completion boundary** of a flow,
where unresolved modes must be absorbed, reflected, dissipated, or leaked. The object:
`ICG = (X, ∂X, Φ, H, R, Q, K, L)` — state space, completion boundary, flow, capacity,
reflection, balance `Q=H−R`, normalised reflection `K=H^{−1/2}RH^{−1/2}`, leakage `L`.

**Stability ⟺ capacity dominates reflection:**
> `Q = H − R ≥ 0`  ⟺  `R ≤ H`  ⟺  `‖K‖ ≤ 1`  (no leakage / no over-reflection).

**HONEST CAVEAT — the unification is generic, not deep.** This three-line algebra
(`Q≥0 ⟺ ‖K‖≤1`) holds for **any** PSD `H` and symmetric `R` — verified: a random PSD `H`
+ random symmetric `R` satisfies it too (`min eig Q=0.44`, `‖K‖=0.12`, identity holds).
So the "universality" across RH / black holes / brains is the **ubiquity of "stability = a
positive quadratic form = a contraction"** — a real and useful organizing pattern, but
**not new mathematics and not a discovery**. The content in each domain is *which* `H`, `R`
are forced by that domain's structure, and whether `R ≤ H` is a theorem there. The
framework is a **reusable language for boundary stability, not a proof by analogy.**

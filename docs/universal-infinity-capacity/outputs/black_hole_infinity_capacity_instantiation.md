# Black-hole instantiation (concrete)

`ICG_BH`: `X` = exterior wave region; `∂X` = event/scattering horizon; `Φ` = radial wave
propagation; `H` = absorbed flux (greybody) capacity; `R` = reflected flux; `Q` = net
absorbed flux; `K` = reflection coefficient; `L` = superradiance.

Verified pattern (Pöschl–Teller, horizon WO): `|R|²+|T|²=1`, `F_abs=|T|²≥0 ⟹ |R|²≤1`
(positive flux ⟺ contraction). **Superradiance** = the exception where rotational/charge
energy extraction makes `|R|²>1` (over-reflection) — the analogue of `‖K‖>1`. This is
standard scattering physics; concrete, not analogy.

Shared: the **form** `Q=H−R≥0 ⟺ ‖K‖≤1`. Domain-specific: *which* `H,R` and whether the
bound is a theorem (RH: open; BH: physics; brain/VFD: analogy). The shared form is generic
(holds for any PSD `H`+sym `R`), so the table is an **organizing map**, not a derivation.

**1. Claim Audit**
- [paper-xxvii.tex:40] “A global balance law `\sum S(e) = const` provides a discrete conservation principle.” Not established. Proposition 2 proves only `\sum_e (S(e)-\bar S)=0` [169-178], which is an algebraic identity after defining `\bar S`; it is not a dynamical conservation law, and it is not the same statement as `\sum S = const`.
- [paper-xxvii.tex:107] “`S_\Omega` coincides with `K_branch` from Paper XXVI … while `S_J` coincides with `K_geo`.” False. In XXVI, `K_branch(e)=|deg(e)-\bar deg|` [paper-xxvi.tex:136-144], whereas here `S_\Omega(e)=deg(e)-\bar deg` [85-91]. In XXVI, `K_geo(e)=(\beta(e)-\bar\beta)/\bar\beta` [159-168], whereas here `S_J(e)=\beta(e)-\bar\beta` [97-102]. These are not the same quantities.
- [paper-xxvii.tex:138-144] Proposition “Existence of the potential.” The proof is basically valid on a finite connected undirected graph: constants span the kernel and zero-mean data lie in the image. But the paper hides the finite-graph hypothesis needed to make the Fredholm/matrix argument precise.
- [paper-xxvii.tex:169-178] Proposition “Global source balance.” Correct, but trivial. It proves an identity, not a physical conservation theorem.
- [paper-xxvii.tex:184-194] Proposition “Local flow conservation.” The proof sketch is correct as a discrete divergence theorem, but the statement is underspecified: `\partial R` is never defined, and connectedness of `R` is unnecessary.
- [paper-xxvii.tex:225-236] Vacuum example. The conclusion is true for the degree-deviation source on `K_{5,5}`, but the stated argument via `S_\rho` is incomplete. `S_\rho(e;r)` depends on a chosen radius and reference profile [73-79]; neither is fixed here. As written, “all events have the same degree and neighborhood profile, so `S_\rho=0`” only follows for a specific choice such as `V_ref=\bar V`.
- [paper-xxvii.tex:243-269] Bottleneck example. The Poisson solve is correct: `u(a_i)=u(c_j)=7/9`, `u(b)=0` is a valid gauge choice, and `K_pot(b)=392/81`, `K_pot(leaf)=49/81` are correct. The arithmetic checks out.
- [paper-xxvii.tex:276-299] Perturbation example is not established. It attaches `f` to `(0,0)` and `(0,4)` [277], but `(0,0)` is not in the first two pentagon layers used in XXIII/XXV; those layers are `L_1={(k,k-1)}` and `L_2={(k,k-2)}` [paper-xxiii.tex:269-277; paper-xxv.tex:272-283]. No solution of `\Delta_G u=S-\bar S` is actually computed, so “gives a nontrivial potential landscape” [288] and “measurable curvature response” [343] are assertions, not results.
- [paper-xxvii.tex:200-219, 267, 372-383] The paper repeatedly claims a source-to-geometry-to-geodesic law. That is overclaimed. It defines a source and solves a graph Poisson equation for `u`; it does not define any modified metric, edge weights, or path rule by which geodesics depend on `u`. At most the toy examples show correlation, not a derived propagation law.

**2. Internal Consistency**
- [paper-xxvii.tex:107] says all explicit toy-model solutions use `S_\Omega`. That is false: the vacuum example uses `S_\rho` [228], the bottleneck uses `S_\Omega` [246-248], and the perturbation example again uses degree deviation [279-288].
- [paper-xxvii.tex:202] says “Source content `S(e)` measures local event-density deviation.” That conflicts with the three-source setup and with the actual worked examples, which use degree deviation, not density.
- [paper-xxvii.tex:246] “Using degree-based density” is terminologically wrong. This is not density; it is the overlap/connectivity source `S_\Omega`.
- [paper-xxvii.tex:40] abstract says `\sum S = const`; [317] table correctly says `\sum(S-\bar S)=0`. The paper does not use one conservation statement consistently.
- [paper-xxvii.tex:277] “Layer-0 event `(0,0)` and Layer-1 event `(0,4)`” conflicts with the actual pentagon layer labels inherited from XXIII/XXV, where `(0,4)` is in `L_1` and `(0,0)` is not in the first two-layer `K_{5,5}` truncation at all.
- Internal `\ref`/`\eqref` usage appears semantically consistent on inspection. The problem is not broken cross-reference targets; it is that the surrounding prose overstates what those referenced items prove.

**6. Top Three Fixes**
1. Repair or delete the perturbation example [276-299]. As written it is tied to the wrong pentagon events and contains no actual Poisson solve. This is the clearest substantive failure.
2. Remove the false source-curvature identifications and the unproved geodesic-law rhetoric [107, 148-152, 200-219, 267, 372-383]. Right now the paper claims more than it shows.
3. Restore the inherited conditionals and clean up the source definitions [73-107, 138-144, 228-248, 358-383]. Specify the radius/reference for `S_\rho`, state the finite-graph hypothesis in Proposition 1, and stop erasing the caveats from XXIII/XXV/XXIV.

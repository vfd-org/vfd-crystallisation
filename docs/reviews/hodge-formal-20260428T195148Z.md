**1. Claim Audit**
- `hodge-formal.tex:195-199`: “`\mathbf{H_{4a}^{alg}/\Omega} ⇒ \mathbf{H_{4a}^{alg}/X}` holds whenever `\iota_X` pulls back algebraic cycles...”  
  Not established. Pulling back algebraic cycles preserves algebraicity, but does not show every relevant class on `X` is a pullback from `\Omega`. You need a surjectivity/image hypothesis.

- `204-213`: “Then ... the Hodge Conjecture for `X` holds.”  
  Only true if `H_{4a}^{alg}/X` is read as already containing the needed algebraicity. As a reduction “via pullback,” the proof is missing the same surjectivity step.

- `340-342`: “The cascade closure functional `F` is `\sigma`-invariant.”  
  No proof or exact local theorem citation. The current foundations material treats F5 conditionally and in a more specific form, not as this bare theorem.

- `349-400`: coefficientwise `\sigma` action and fixed part.  
  The rational version is fine. The integral lemma proof is too casual: it assumes a basis/tensor decomposition and ignores torsion-module bookkeeping. State the `\mathbb Q(\varphi)` result as primary.

- `425-455`: `\mathbb{CP}^n` cohomology and Hodge.  
  Proof establishes the claim.

- `472-509`: flag varieties.  
  Conclusion is classical, but the proof’s Schubert degree indexing is wrong/ambiguous: closures of Bruhat cells of dimension `\ell(w)` do not directly define cohomology classes in degree `2\ell(w)` unless opposite/codimension conventions are specified.

- `528-535`: Danilov theorem.  
  False as stated. `H^{2p}` rank is not “the number of `p`-dimensional cones”; e.g. for `\mathbb{CP}^2`, three rays do not give rank three in `H^2`. Generation by torus-invariant cycles is the usable statement.

- `541-558`: toric Hodge theorem.  
  The conclusion is true for smooth complete toric varieties, but the proof relies on the false rank statement above. Repair the Danilov citation/presentation.

- `572-602`, `588-592`: CM abelian status.  
  Correctly demoted: absolute Hodge and Hodge-locus algebraicity do not prove Hodge algebraicity in general.

- `624-631`: Lefschetz hyperplane theorem.  
  Acceptable.

- `635-664`: complete intersections.  
  Overclaim. Lines `650-659` assert primitive middle cohomology is handled for codimension `1` and `2`; Noether-Lefschetz does not prove this general statement. This theorem must be replaced by a limited status statement.

- `677-703`: cascade-embeddable proposition.  
  Same flaw as `195-213`: `\Omega`-algebraicity plus regular pullback is insufficient without showing the fixed classes on `X` come from `\Omega`.

- `713-733`: Langlands-cascade reduction.  
  Not theorem-grade. The proof sketch assumes motive automorphy, cascade realization, and an embedding construction without establishing any of them.

- `786-802`: 600-cell spectrum.  
  The local spectrum script reproduces `94/120 = 78.3%` and the paired eigenvalues. But the cited script uses numerical eigendecomposition; exactness needs the Paper XXII representation-theoretic table, not just this script.

- `815-836`: finite-noise diagnostic.  
  Source supports the setup, but R2/R3 equal `1.0000` because `P_fix` is explicitly inserted. This is a diagnostic, not independent evidence for an emergent closure projector. My fresh run did not finish in this environment.

**1. Claim Audit**

- Lines 174-202, “cascade algebraicity bridge axiom”: the implication at lines 195-199 is not established. Regularity of `\iota_X` only says algebraic cycles pull back to algebraic cycles. It does not say every `\sigma`-fixed class on `X` is the pullback of a `\sigma`-fixed class on `\Omega`. You need a surjectivity/lifting hypothesis. Also `\Omega` is not defined here as a smooth/projective algebraic variety with cycle theory.

- Lines 204-217, “Main conditional reduction”: the stated proof is absent. If `H_{4a}^{alg}/X` is assumed, the conclusion is essentially assumed. If only the substrate version is assumed, the missing pullback-surjectivity problem above breaks the argument.

- Lines 346-348, “The cascade closure functional `F` is `\sigma`-invariant”: no definition of `F`, no proof, no citation. This cannot stand as a theorem.

- Lines 355-364, “`\sigma` action on cohomology”: coefficientwise action is fine over `\Q(\varphi)`, but the integral statement and “commuting with the Hodge decomposition” need qualification. Hodge decomposition is not a decomposition of `H^*(X,\Z[\varphi])`.

- Lines 382-407, “`\sigma`-fixed classes are rational”: true as coefficient linear algebra over `\Q(\varphi)`. The integral proof is too basis-dependent and should be rewritten modulo torsion/free-part issues.

- Lines 431-435 and 442-462, `\CP^n`: proof establishes the classical claim.

- Lines 478-492, Bruhat/Schubert claims: true only with standard hypotheses on `G` and `P`. The degree indexing is confused: if `C_w \cong \C^{\ell(w)}`, then `\overline C_w` has dimension `\ell(w)`, not cohomological degree `2\ell(w)` unless you use dual/opposite Schubert classes.

- Lines 534-543, Danilov theorem: the rank claim is false as stated. For example `\CP^2` has three one-dimensional cones but `H^2(\CP^2,\Z)` has rank 1. The generation-by-orbit-closures part is enough for the later Hodge argument.

- Lines 578-599, CM abelian status: citation-only claim is acceptable as a weak status statement. But lines 610-615 overclaim: arbitrary CM fields do not carry a canonical `\Z[\varphi]` action, and cascade-embeddability is not proved.

- Lines 630-637, Lefschetz hyperplane theorem: acceptable.

- Lines 641-670, complete intersections: serious overclaim. The theorem says only outside middle degree plus selected middle cases, but the proof concludes full Hodge for all low-codimension complete intersections. Lines 654-666 assert primitive middle cohomology is explicitly algebraic; this is not established and is generally the hard part.

- Lines 683-709, cascade-embeddable proposition: same defect as the main reduction. Pullback of algebraic classes is not enough; one needs every relevant class on `X` to come from `\Omega`, or must assume `H_{4a}^{alg}/X` directly.

- Lines 713-715, universality conjecture: fine as conjecture, but it duplicates `H_{4a}` and should use the same category of objects.

- Lines 719-739, Langlands-cascade reduction: not proved. “Every motive is automorphic” is conjectural, and “realizes the motive” does not imply an embedding of `X` into a cascade substrate. This contradicts lines 902-906, which say no Langlands-cascade theorem is asserted.

- Lines 792-808, 600-cell spectrum: I ran the cited spectrum script; it verifies the displayed spectrum and `94/120 = 78.3%`. But the script uses numerical eigendecomposition plus recognition; exactness depends on the closed-form source, not on the script alone.

- Lines 821-842, finite-noise diagnostic: source text matches the quoted values, but the dynamics script did not finish in this sandbox. Also line 830 calling R3 a “control” is misleading; the closure projector forces the result by construction.

- Lines 846-854, aria H4O: not locally verified. The cited `aria-chess/...` artifact is outside this repository.

- Lines 910-918, “Establishing cascade universality ... would complete the proof”: false as written. The paper itself says `H_{4a}^{alg}` is also required.

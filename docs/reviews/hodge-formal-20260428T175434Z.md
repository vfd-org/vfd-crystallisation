Line numbers below refer to `papers/millennium-hodge-formal/hodge-formal.tex`.

1. **Claim Audit**

- “Every σ-fixed, Q-rational class in `H^*(Ω,Q(φ))` is algebraic” (171-184): overbroad even as an axiom unless restricted to even rational Hodge classes. Algebraic cycle classes do not live in arbitrary cohomological degree.
- “Main conditional reduction” (195-204): not proved. If `H_{4a}^{alg}/X` is assumed, the proposition is essentially the Hodge Conjecture restated. If only `H_{4a}^{alg}/Ω` is assumed, the missing step is surjectivity/lifting: not every class on `X` is shown to be a pullback from `Ω`.
- “σ acts on cohomology” (340-349): coefficientwise action is fine over `Q(φ)`. Integral proof assumes a tensor/basis decomposition not justified in the presence of torsion.
- “σ-fixed classes are rational” (367-391): true over `Q(φ)`, but the proof should use `α+βφ` decomposition, not a basis of integral cohomology. Also `Q[φ]`/`Q(φ)` notation is inconsistent.
- “F5: closure functional `F` is σ-invariant” (331-333): not established here and not consistent with the newer Paper XXXVI formulation, where F5 is conditional factor-2 equality under H-SIG, not unconditional σ-invariance.
- “CP^n admits a Kähler embedding into a cascade substrate rung” (398-420): proof does not establish the claim. The `n≥3` argument “CP^n can be realized as a projective bundle over CP^{n-1}” is false as stated.
- “Hodge for CP^n” (435-454): proof is adequate, conditional on the classical cohomology statement.
- “Schubert classes span cohomology” (483-508): theorem is classically true after correcting hypotheses and degree conventions. As written, `[\bar C_w] ∈ H^{2ℓ(w)}` uses the wrong indexing unless opposite cells/Poincare duals are specified.
- “Danilov” theorem (527-535): false as stated. `H^{2p}` rank is not the number of `p`-dimensional cones; e.g. `P^2` has three rays but `rank H^2=1`. Generation by invariant cycles is the usable true part.
- “Hodge for smooth complete toric varieties” (540-557): proof works only after replacing the false rank claim and restricting to the projective/Hodge-Conjecture setting.
- “CM abelian varieties” (571-591): Deligne gives absolute Hodge, not algebraicity. The later theorem correctly says this does not imply Hodge in general.
- “Selected complete intersections” (634-663): not proved. The middle primitive cohomology argument is handwaving; Noether-Lefschetz does not prove Hodge for all codimension `≤2` complete intersections.
- “Hodge for cascade-embeddable” (676-696): same gap as the main proposition. Pullback from `Ω` is treated as if it were surjective onto relevant classes on `X`; this is not shown.
- “Cascade universality” (706-708): acceptable as a conjecture, but “smooth projective Kähler manifold” mixes algebraic and analytic categories.
- “Langlands-cascade reduction” (712-732): not a theorem. Even motivic automorphy would not by itself construct a cascade embedding.
- Numerical spectrum claim `94/120 = 78.3%` (785-801): rerunning the cited script confirms the count. The text’s “computed in Q(√5) arithmetic” is stronger than the script evidence, which uses numerical eigendecomposition plus recognition.
- Dynamics values `0.6639`, `1.0000`, `1.0000` (814-835): rerunning the cited script confirms these values.
- H4O witness (839-847): not locally verified. The cited `aria-chess/...` paths are not present in this repository.

2. **Internal Consistency**

- The abstract says five classes have Hodge “already known classically” (57-66), including CM abelian varieties and low-codimension complete intersections. Later the paper says CM/CI evidence is weaker than “Hodge proved” (746-759). These cannot both stand.
- `thm:main-cond` is a `proposition` (195) but is repeatedly called “Theorem” (48, 223, 225).
- Scope remark says the main result is “strictly conditional on H_{4a}” (234-236), but the proposition also assumes `H_{4a}^{alg}` (197-199).
- `Ω`, `Ω_r`, finite `E8` root substrate, and algebraic variety/scheme are used interchangeably without a definition that supports cycle pullback (166-190, 283-290, 323-329).
- Internal refs appear to resolve, but several citation keys do not: `Voisin2002`, `Hodge1941`, `BorelHirzebruch`, `Danilov`, and `CDK`.

3. **External Consistency**

- `CascadeFoundations` does contain `H_{4a}` as an open embeddability/compatibility hypothesis, not algebraicity. This supports the paper’s demotion of `H_{4a}` alone, but not the main reduction.
- Paper XXXVI contradicts the local F5 statement: current F5 is conditional factor-2 equality for σ-fixed closure fields under H-SIG, not “F is σ-invariant.”
- `docs/convergence-with-smart.md` and `rh-formal.tex` support only descriptive “Bridge Axiom Pattern” language. The current text mostly respects that.
- The spectrum and dynamics scripts support the numerical values quoted, except for the “exact arithmetic” wording.
- I could not verify the claimed `aria-chess` H4O covariance-pair artifact locally.

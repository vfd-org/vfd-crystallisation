**2. Internal Consistency**

- Bridge names conflict: abstract uses `H_BSD-Conv`/`H_BSD-Spec` [44-53], later the paper uses `B_BSD,conv`, `B_BSD,rank`, `B_BSD,spec` [314-336], and Section 573 omits the rank bridge.

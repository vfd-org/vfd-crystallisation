**1. Claim audit**

- “The set of events carries a natural strict partial order given by birth-angle comparison” ([paper-xxiii.tex](/mnt/c/Users/nexus/OneDrive/Documents/My Projects/vfd-crystalisation-paper/papers/paper-xxiii/paper-xxiii.tex:43), formalised at ll. 158-163). The manuscript does **not** establish this as an intrinsic structure of the geometry. The order depends on choosing the interval `[0,2\pi)` in Definition 5 (ll. 135-139), i.e. on an arbitrary origin for a periodic phase. Shift the zero of `\theta`, and birth angles change; the induced order can change as well. What is proved is at best an order relative to a chosen phase gauge, not a “natural” order derived solely from the event structure.

- Proposition “The relation `\prec` on `\Ecal` is a strict partial order: it is irreflexive, transitive, and non-total” ([paper-xxiii.tex](/mnt/c/Users/nexus/OneDrive/Documents/My Projects/vfd-crystalisation-paper/papers/paper-xxiii/paper-xxiii.tex:171), ll. 171-185). The proof establishes irreflexivity and transitivity. It does **not** establish non-totality for the actual dual-600-cell system. The non-totality paragraph is conditional: “If `\theta_b(e_1)=\theta_b(e_2)`...” (l. 184). No argument is given that such equality occurs in the 600-cell model. A toy pentagon example is not a proof of the main construction.

- Theorem “No canonical global clock” ([paper-xxiii.tex](/mnt/c/Users/nexus/OneDrive/Documents/My Projects/vfd-crystalisation-paper/papers/paper-xxiii/paper-xxiii.tex:207), ll. 207-217). This over-claims twice. First, it relies on the unproved non-totality above. Second, the statement “no unique function `t:\Ecal\to\mathbb R` determined solely by `(\Ecal,\prec)`” is not a mathematical definition; “determined solely” is doing all the work and is undefined. The proof shows only: if incomparable pairs exist, then linear extensions are non-unique. That is weaker.

- Proposition “If a set of `k` events share the same birth angle, then there exist at least `k!` distinct observers...” ([paper-xxiii.tex](/mnt/c/Users/nexus/OneDrive/Documents/My Projects/vfd-crystalisation-paper/papers/paper-xxiii/paper-xxiii.tex:234), ll. 234-240). Conditionally fine. But again, the paper never proves that equal-birth layers occur in the 600-cell system. As a result, this is only a conditional combinatorial observation, not evidence about the main model.

- Worked example, second event layer ([paper-xxiii.tex](/mnt/c/Users/nexus/OneDrive/Documents/My Projects/vfd-crystalisation-paper/papers/paper-xxiii/paper-xxiii.tex:261), ll. 261-263). As written it is false under the paper’s own definitions. With overlap defined by a strict inequality `|\alpha_k-\beta_l(\theta)|<\delta` (l. 255) and birth angle defined as an infimum (l. 138), the pair `(k,k-1)` is born at `2\pi/5-\delta`, not at `2\pi/5`. Accordingly l. 262 is also false: new pairs do become admissible for `\theta\in(2\pi/5-\delta,2\pi/5)`. The example does not instantiate the formalism correctly.

- “At each discrete alignment angle, up to 120 simultaneous events occur” ([paper-xxiii.tex](/mnt/c/Users/nexus/OneDrive/Documents/My Projects/vfd-crystalisation-paper/papers/paper-xxiii/paper-xxiii.tex:273), l. 273). Unsupported. No 600-cell calculation is given.

**5. Surface issues**

- `Foundation~C2.bis` is cited repeatedly in XXI, but I cannot find that label in the local `paper-xxxvi.tex`. At minimum this is a repository-internal citation failure.

- Route B repeatedly conflates a scalar functional `F`, a graph Laplacian `\Delta_{H_4}`, and Stone’s generator `A`. That is not merely exposition; it is a notation error that infects the logic.

- Capitalisation/style are mostly controlled. The main surface problem is not typo-level; it is that theorem statements are written as if upstream assumptions had already been discharged.

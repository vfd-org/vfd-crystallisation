**Claim Audit**

- Line 134: “Cascade-Hecke reformulation of BSD, conditional only on H\(_{\text{BSD-Spec}}\)” is not a theorem. H\(_{\text{BSD-Spec}}\) at lines 51-53 already asserts both identities needed to get BSD. The proof is tautological unless those two identifications are independently proved, which they are not.
- Line 207: “Termwise operator bound” is not proved as stated. The proof obtains a constant \(2\varphi\log p\), then “absorbs” \(\varphi\) into \(2\log p\), which is false. Worse, \(v_\varphi(p)\) is undefined/trivial because \(\varphi\) is a unit in \(\mathbb Z[\varphi]\).
- Line 239: absolute convergence for \(\mathrm{Re}(s)>1\) is plausible only after correcting the constant and specifying the operator space/norm. It does not discharge convergence at \(s=1\).
- Line 266: the Rankin--Selberg continuation claim is not established. Cauchy--Schwarz does not produce a meromorphic operator-valued continuation of \(T_E(s)\), and defining \(T_E(1)\) as either a residue or a value is not a well-defined regularisation.
- Line 304: Eichler--Shimura is imported but misstated. The displayed \(\mathbb Q\)-isomorphism mixes singular cohomology of \(E\) with complex modular-form spaces; this needs a precise modular-curve/Jacobian formulation.
- Line 314: “rank-\(f_E\)-eigenvectors correspondence” is false. Mordell--Weil rank is not the dimension of an eigenspace in \(H^1(E,\mathbb Q)\), whose dimension is fixed.
- Line 320: “\(\ker T_E = E(\mathbb Q)\otimes\mathbb Q\)” is not proved and is generally impossible on \(H^1(E,\mathbb Q[\varphi])\), whose dimension is bounded while elliptic-curve ranks are not. This must be a bridge axiom, not a theorem.
- Line 346: depends entirely on the false Theorem 320.
- Line 369: the spectrum-to-\(L\)-function theorem is unsupported. The cited “cascade trace formula” is not present locally; the proof only sketches placeholders. Lines 421-424 also confuse the normalized critical point \(1/2\) with the BSD central point \(s=1\).
- Line 431: the BSD proof only combines the unproved/false Corollary 346 and Theorem 369.
- Lines 499-505: the rank-0 example is plausible, but the notation “\(\operatorname{ord}_{s=1} L(E,1)\)” is wrong; order applies to \(L(E,s)\), not the number \(L(E,1)\).
- Lines 521-527: the rank-1 example is asserted without conductor, database citation, analytic-rank computation, or script verification.
- Lines 546-564: the 600-cell spectrum count is locally reproduced by the script, but it is irrelevant to BSD unless a separate spectral-to-\(L(E,s)\) bridge is assumed.
- Lines 577-601: the finite-noise diagnostic is a toy projector computation. It does not support \(H_{\text{BSD-Spec}}\) except as analogy.
- Lines 603-613: the aria H\(_4\)O artifact is not in this repository; I could not verify it locally.
- Lines 640-647: “handles ALL ranks uniformly” and “closes BSD in one stroke” are false relative to the paper.

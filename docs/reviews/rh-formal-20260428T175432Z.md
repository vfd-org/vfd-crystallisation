**1. Claim Audit**

Established as stated, with the advertised scope:

- “Cascade zeta identity” (`rh-formal.tex:400`): proved only because `\zetacas` is defined as classical `\zeta` (`1165-1174`). This is not a derivation, but the theorem now says that.
- “Critical line as fixed set” (`415`): proof establishes the fixed set of `\tau_{\mathrm{crit}}=s\mapsto1-\bar s`, not of holomorphic `\sigma` alone. Correctly scoped.
- “Conditional critical-line theorem” (`465`): valid but tautological; `H_{\sigma-fix}` already says `\rho=1-\bar\rho` (`1569-1576`), i.e. RH.
- “Pentagonal torsion” (`701`), “K-multiset” (`762`), and “Exact closed form” (`798`): the cited scripts reproduce the stated values. I ran `derive_pentagonal_clock_B8p_proper.py`; it returns `{0:1,20:5,52:5,72:1}` and the stated `z^10` coefficient.
- “Mellin pole-symmetry” (`836`): the algebraic palindromy and real-part staircase are established. I ran `derive_pentagonal_clock_B11_mellin.py`; it matches `1/2 ± K/10`.
- “Splitting law” (`644`), `\eta_p` convergence (`1136`), Euler product (`1212`), functional equation (`1242`), analytic continuation (`1285`), `\zeta_K` identification (`1374`), and Dedekind factorization (`1390`) are standard/definitional and proved or imported correctly, except for the character-uniqueness error below.
- “Galois action on spectrum, conditional” (`1501`): proof establishes only the `\mathbb Q(\varphi)`-part under H-Mat. This is correctly limited.
- “Spectral sigma-attractor decomposition” (`1704`): verified against Paper XXII and by running `test_sigma_attractor_spectrum.py`.
- “Mode suppression by closure flow” (`1835`): proof establishes suppression for the explicitly defined polynomial flow. It does not establish that this flow is the original cascade closure functional.

Not established or overstated:

- `rh-formal.tex:440-441`, `1391-1393`: “unique non-trivial Dirichlet character modulo 5” is false. There are three non-principal primitive characters modulo 5. Correct statement: “the unique non-trivial quadratic character modulo 5.”
- `1818-1832`: “canonical for the cascade Galois data alone” is too strong. The polynomial filter is chosen to vanish on the rational eigenvalues; uniqueness needs extra normalization/minimal-degree restrictions and is not proved as a cascade principle.
- `1906-1923`, `2044-2048`: saying the theorem “establishes the discrete finite-dimensional content” of `H_attr` is only true for the engineered flow `e^{-t p_{\rm fix}(L)^2}`. It does not validate a physical or pre-existing closure dynamics.
- `1951-1983`: `H_trace` is not mathematically well-formed. It evaluates a strip-holomorphic `h(s)` at Laplacian eigenvalues `\lambda`, uses `K` as both set and variable, and leaves `K^*`, `\lambda_K^\pm`, and `h_K^{pair}` undefined.
- `2007-2008`: “unique `\tau_{\rm crit}`-fixed point” is false unless explicitly restricted to the principal real branch. The `K=0` pole has imaginary periodic copies, all on `Re(s)=1/2`.

**2. Internal Consistency**

- Cross-references resolve. I found no broken `\ref`/`\eqref`.

- The paper is not consistent about **state-completeness vs universal completeness**. Theorem `\ref{thm:noncontextuality}` uses the former in the hypotheses but writes a universal quantifier in the conclusion ([paper-xxx.tex](/mnt/c/Users/nexus/OneDrive/Documents/My Projects/vfd-crystalisation-paper/papers/paper-xxx/paper-xxx.tex:596), line 599).

In prior papers it denotes the closure manifold / zero set of `\Fcal`; here `\Mcl(\Ocal)` is a subset of `\mathbb{CP}^{N-1}` ([paper-xxx.tex](/mnt/c/Users/nexus/OneDrive/Documents/My Projects/vfd-crystalisation-paper/papers/paper-xxx/paper-xxx.tex:413), lines 413-419). The manuscript knows the typing issue, but the notation still invites confusion.

Those are not the same claim.

- Route C oscillates between configuration-sector notation `\Phi_i(\Ocal)` and projector notation `\Pi_i` without always saying which object the probability is attached to. The notation is mostly manageable, but not clean.
